AI 채팅창에 표시된 기능 버튼 가이드

AI 채팅의 숨겨진 강력한 기능들을 만나보세요. 이 가이드는 파일 추가부터 가이드 학습까지, AI의 다양한 기능들을 쉽고 명확하게 설명하여 여러분이 AI를 100% 활용할 수 있도록 돕습니다.

① 파일 추가

사용자가 파일을 채팅창에 업로드 한 후 질문을 하게 되면, AI는 업로드된 파일의 내용을 분석하고, 그 내용을 바탕으로 질문에 답하거나 자료를 요약하는 등의 작업을 수행할 수 있습니다. 심지어 프로그래밍 코드 파일을 올리고, 코드 분석까지도 가능합니다.

파일 업로드 방법 2가지

💻 내 컴퓨터에서 직접 업로드

사용자의 PC 에 저장된 문서나 이미지 파일을 채팅창에 바로 올리는 가장 기본적인 방법입니다.

☁️ 구글 드라이브에서 파일 추가

구글에서 제공하는 클라우드 서버에 저장된 파일들을 불러와 AI에게 분석을 요청하는 방식입니다.

🚀 직접 해보자!

  • 문서 파일을 업로드 한 후 채팅창에 질문 해보기

② 딥 리서치 기능

'딥 리서치'는 AI가 이미 학습한 지식이나 단편적인 웹 검색을 기반으로 답변하는 일반적인 대화와 달리, 복잡하고 깊이 있는 주제에 대해 AI가 웹상의 방대한 정보를 직접 탐색하고 분석하여 종합적인 보고서 형태로 답변을 제공하는 강력한 기능입니다. 사용자가 직접 여러 웹사이트를 방문하며 정보를 찾고 정리해야 하는 수고를 덜어줍니다.

🌐 한 곳이 아닌 여러 곳을 샅샅이 뒤진다

질문한 주제에 대해 딥리서치 기능을 사용한 AI는 수많은 웹사이트와 전문 자료를 실시간으로 탐색하고 핵심 내용을 분석합니다. 이때 '전문 자료'란 공개적으로 접근 가능한 학술 논문, 연구 보고서, 공신력 있는 언론 기사, 기술 문서 등 광범위한 정보를 포함합니다.

⚙️ 딥 리서치 사용 절차

1. 질문 입력
→
2. 리서치 계획 수립/승인
→
3. AI 웹 탐색/분석
→
4. 종합 보고서 생성

💡 딥리서치 질문 입력 잘하는 방법

딥 리서치 기능은 사용자의 질문 의도를 얼마나 정확하게 파악하느냐에 따라 보고서의 품질이 크게 달라집니다. 아래의 요령들을 활용하여 AI가 사용자의 생각을 더 잘 이해하도록 도와주세요.

요령 1 : '비교'하고 '분석' 해달라고 요청하기

안 좋은 예시: "전기차에 대해 알려줘."

좀 더 좋은 예시: "전기차와 하이브리드 자동차의 장점과 단점을 비교 분석해주고, 유지비는 어느 쪽이 더 저렴한지 알려줘."

요령 2 : '상황'과 '조건'을 구체적으로 알려주기

안 좋은 예시: "노후 준비는 어떻게 해야 해?"

좋은 예시: "현재 60대 후반인 부부가, 서울에서 거주하면서, 월 200만 원으로 생활한다고 가정했을 때, 가장 안정적인 노후 준비 방법에는 어떤 것들이 있을까?"

요령 3 : '여러 관점'이나 '다양한 의견'을 모아달라고 하기

안 좋은 예시: "금리 인상이 경제에 미치는 영향"

좋은 예시: "미국이 금리를 인상하면 한국 경제에 어떤 영향이 있을지, 긍정적인 측면과 부정적인 측면을 나누어 여러 경제 전문가들의 의견을 종합해서 설명해줘."

요령 4 : 보고서의 구조를 질문 창에 기본적으로 짜주기

"텍사스인스투르먼트(TXN) 종목에 대하여 주식투자 입문 보고서를 만들어줘. 보고서 구조는 아래와 같이 작성해줘. 1. 기업 개요, 2. 주요 사업 부문..."

③ 이미지 생성, 수정 요청하기

채팅창에 텍스트로 원하는 이미지에 대해 설명하면, AI가 그 설명을 바탕으로 새로운 이미지를 만들어주는 기능입니다.

- 이미지 생성을 시도해보자

가장 기본적인 사용법으로, 만들고 싶은 이미지의 내용을 간단한 문장으로 설명하여 요청하는 방식입니다. AI는 사용자의 설명을 바탕으로 기본적인 이미지를 생성해 줍니다.

"마당에 감나무가 있는 한옥집을 그려줘."

- 좀 더 구체적인 명령어로 이미지 생성하기

원하는 결과물을 더 정확하게 얻기 위해, 이미지에 포함될 요소들을 상세하게 설명하는 방식입니다. 배경, 사물의 색상, 구도, 인물의 표정 등 구체적인 정보를 추가할수록 상상에 가까운 결과물을 얻을 수 있습니다.

"저녁노을이 지는 하늘 아래, 마당에 감나무가 있는 한옥집 툇마루에 할머니와 손주가 앉아있는 모습을 그려줘"

- 이미지 스타일 요청하기

단순히 내용을 설명하는 것을 넘어, 원하는 그림의 스타일이나 화풍을 지정하여 요청하는 방식입니다. 사진, 만화, 수채화, 유화, 특정 화가의 스타일 등 다양한 스타일을 적용할 수 있습니다.

"반 고흐의 화풍으로, 별이 빛나는 밤의 서울 풍경을 그려줘."

- 이미지 수정 요청하기

AI가 이미 생성한 이미지를 보면서, 마음에 들지 않는 부분을 수정하도록 요청하는 기능입니다. 색상을 바꾸거나, 특정 사물을 추가 또는 제거하는 등 대화를 통해 결과물을 완성해 나갈 수 있습니다. 내가 직접 사진 파일을 올려 수정해달라고 요청하는 것 또한 가능합니다.

"이 사진 표정을 웃는 모습으로 바꿔줘"

④ 동영상

텍스트(프롬프트)로 원하는 영상의 내용을 상세히 설명하거나, 기존에 가지고 있는 사진 파일을 업로드하여 AI에게 짧은 동영상을 만들어달라고 요청하는 기능입니다. 머릿속 아이디어를 생생한 영상으로 빠르게 구현할 수 있습니다.

※ 참고: 현재 AI 기술로는 주로 8초 내외의 짧은 영상(클립) 생성이 가능합니다. 또한 제미나이는 채팅창에 동영상 버튼이 있지만, GPT는 왼쪽 사이드바에 'Sora'라는 버튼을 눌러야 동영상 생성이 가능합니다.

- 텍스트(프롬프트)를 입력하여 동영상 만들어보기

만들고 싶은 영상의 장면, 등장인물, 분위기, 카메라 움직임 등을 글로 상세하게 묘사하여 요청합니다.

"황금 시간대에 드론으로 촬영한, 파도가 부서지는 캘리포니아의 빅서 해안 풍경 영상"

- 사진 파일을 업로드 하고, 사진을 동영상으로 바꿔보기

가지고 있는 사진을 AI에게 보여주고, 이 사진에 생동감을 불어넣어 영상으로 만들어달라고 요청할 수 있습니다.

"(가족 사진을 올리고) 이 사진 속 인물들이 부드럽게 미소 짓고, 배경의 나뭇잎이 바람에 살짝 흔들리는 영상을 만들어줘."

"→ 부모님으로부터 돌아가신 할아버지나 할머님 사진을 받고, AI를 활용하여 사진을 영상으로 만들어 부모님께 드린다면, 부모님이 감동을 받을 수 있겠다는 생각을 해본 적이 있습니다."

⑤ Canvas (캔버스)

캔버스(Canvas)는 AI와 사용자가 함께 글이나 코드 같은 결과물을 만들어나가는 **'디지털 작업판'**이자 **'협업 공간'**입니다. 물론 일반 채팅창에서도 AI의 답변을 수정하고 대화를 이어갈 수 있지만, 본격적으로 결과물을 다듬고 싶을 땐 이 '작업판' 틀 안에서 AI와 소통하는 것이 훨씬 편리합니다. 캔버스에서는 AI가 만든 초안을 바탕으로 사용자가 직접 수정하거나, AI에게 대화를 통해 계속해서 편집을 요청하며 결과물을 함께 완성해 나가는 모든 과정이 이루어집니다. 마치 화가가 캔버스 위에서 그림을 그리고 고쳐가며 작품을 완성하듯, 사용자는 이 디지털 캔버스 위에서 AI와 함께 내용을 더 구체화하고 완성도를 높여나갈 수 있습니다. 또한 캔버스에서 만들어낸 작업물은 바로 문서 파일로도 저장이 가능합니다.

📝 문서나 보고서를 만들 때

AI가 써준 초안을 보면서 "이 부분은 좀 더 자세히 설명해줘" 또는 "표로 정리해줘" 와 같이 요청하며 글의 완성도를 높일 수 있습니다.

💻 프로그래밍 코드를 짤 때

AI가 만들어준 코드를 실행해보고, 오류가 발생하면 그 내용을 캔버스 안에서 바로 수정해달라고 요청할 수 있습니다.

🚀 직접 해보자! (예시를 통한 연습)

1. 캔버스 버튼을 누르고, "추석 연휴 2박 3일 일본 여행 계획'이라는 제목으로 캔버스 창을 띄워줘." 라고 입력

2. → 캔버스 내 채팅창에 "2박 3일 여행코스를 짜서 내용을 추가해줘" 라고 입력

3. → 답변을 받은 뒤 추가 명령어를 입력하여 추가 편집

✨ 추가 기능 (Gemini)

구글 제미나이는 캔버스 내용을 자동으로 인포그래픽, 오디오 요약 등으로 변환시켜주는 기능도 있습니다. (캔버스 작업 화면 오른쪽 상단 '만들기' 버튼)

⑥ 가이드 학습

'가이드 학습'은 AI가 정답을 바로 알려주는 대신, 사용자가 특정 주제를 깊이 있게 이해할 수 있도록 대화형 질문을 던지고 단계별로 개념을 설명하며 학습 과정을 이끌어주는 개인 맞춤형 학습 도우미 기능입니다. 마치 1:1 과외 선생님처럼, AI는 사용자의 이해도를 확인하며 진도를 나가기 때문에 혼자 공부하기 어려운 개념을 익히거나 문제 해결 과정을 배울 때 매우 효과적입니다.

특징 ① : 쌍방향 소통으로 학습이 진행

→ 질문에 대한 답변으로써 끝나는 일반 채팅과는 달리, 가이드 학습 모드에서 AI는 '학습 촉진자(Facilitator)' 또는 '코치'의 역할을 합니다. AI가 학습 과정을 설계하고, 먼저 질문을 던지며 사용자의 적극적인 참여를 유도하는 등 끊임없이 상호작용하며 학습을 진행합니다.

특징 ② : 내용을 끊어서 차근차근 단계단계 배우는 것이 가능

→ 복잡하고 어려운 주제라도 AI가 핵심 개념을 작은 단위로 나누고, 하나씩 차근차근 설명해줍니다. 사용자는 각 단계를 완전히 이해한 후에 다음으로 넘어갈 수 있어 학습에 대한 피로도가 적습니다.

특징 ③ : 가이드 학습을 음성 대 음성으로 주고 받는 것도 가능

  • 데스크탑: 마이크 버튼으로 음성 질문을 입력하면, 텍스트로 변환되어 표시되고 AI 답변은 음성으로 들을 수 있습니다.
  • 모바일: 스마트폰 앱에서는 통화하듯 실시간 음성 대화가 가능합니다. 이는 마치 실제로 사람에게 과외를 받는 듯한 느낌을 줍니다. (음성 대화 버튼 → "가이드 학습으로 OOO 알려줘" 요청)

📚 가이드 학습으로 대화를 나누면 좋은 소재

  • 개념 학습 : "양자역학의 기본 원리가 궁금해", "블록체인이란 무엇인지 차근차근 알려줘" 와 같이 혼자 공부하기 어려운 과학, 역사, 경제 등의 개념을 배울 때 좋습니다.
  • 새로운 기술 배우기 : "파이썬 코딩의 기초에 대해 배우고 싶어", "스페인어 기초 회화를 연습하고 싶어" 처럼 새로운 언어나 기술을 익히는 첫걸음으로 활용할 수 있습니다.
  • 문제 해결 과정 이해하기 : "이 수학 문제를 어떻게 풀어야 할지 단계별로 알려줘" 와 같이 정답보다는 문제에 접근하고 해결하는 과정을 배우고 싶을 때 유용합니다.
