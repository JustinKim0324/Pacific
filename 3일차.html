AI 심화 활용 가이드

AI의 기본 기능을 넘어, 나만의 AI를 만들고, 코딩을 체험하며, AI 기술의 핵심 원리인 API까지 이해하며 AI 활용 능력을 한 단계 업그레이드해 보세요.

① GEMS / GPTS 활용하기

GEMS(Gemini)나 GPTS(ChatGPT)는 반복적인 작업을 위해 매번 긴 프롬프트 입력하는 번거로움을 해결해주는 '나만의 특화 AI' 기능입니다. 특정 역할이나 작업 형식을 미리 설정해두면, 필요할 때마다 짧은 질문만으로 일관성 있고 효율적인 결과물을 얻을 수 있습니다.

  1. 왼쪽 사이드바에서 'GEMS' 메뉴 확인하기 → 'GEMS 탐색하기' 클릭
  2. '+새 Gem' 클릭
  3. '이름' 란에 만들고자 하는 GEMS 이름 짓기, '요청사항' 란에 지침 입력, '지식' 란에 첨부파일 업로드
  4. '오른쪽 미리보기' 란을 통해 테스트 가능
  5. 오른쪽 상단 '저장' 버튼을 누르면 GEMS 생성 완성!
  1. 왼쪽 사이드 바에서 꽃잎4장 문양의 'GPT' 단어 클릭 → 오른쪽 귀퉁이 최상단 ‘+만들기’ 클릭
  2. GPT 와 채팅창에서 대화를 통해 이름, 지침, 설정 등을 입력
  3. 필요시 기초지식을 위한 ‘첨부파일’ 첨부
  4. 오른쪽 화면에서 테스트 가능
  5. 오른쪽 상단 귀퉁이 '만들기' 버튼을 누르면 GPTS 생성 완성!

② AI로 코딩 맛보기

코딩을 전혀 모르는 사람도 간단한 아이디어를 직접 코드로 구현해 볼 수 있게 되었습니다. AI에게 원하는 기능을 말(프롬프트)로 설명하면, AI가 복잡한 코드를 대신 작성해줍니다. 간단한 웹페이지나 인포그래픽, 게임 등을 만들어보세요.

홈페이지는 보통 뼈대를 만드는 'HTML', 디자인을 꾸미는 'CSS', 움직임을 주는 'JavaScript' 코드로 만들어집니다. 우리는 이런 언어를 몰라도 코딩 작업을 할 수 있습니다.

기본 절차:

  1. Canvas 버튼을 누르고 프롬프트를 입력합니다.
  2. AI가 코딩을 하여 홈페이지를 만듭니다.
  3. 완성된 코드를 메모장에 붙여넣기 합니다.
  4. 코드가 포함된 메모장을 .html 확장자 파일로 저장합니다.
  5. 웹호스팅 서비스에 연결하면 누구나 접속할 수 있는 웹페이지 완성!

인포그래픽은 복잡한 정보나 데이터를 시각적인 이미지로 표현하여 쉽고 빠르게 이해할 수 있도록 만든 콘텐츠입니다. 구글 Gemini는 왼쪽 사이드바에 위치한 ‘Canvas(캔버스)’ 창의 내용을 바탕으로 인포그래픽을 자동으로 만들어주는 ‘만들기’ 버튼 기능이 있습니다.

③ API 개념 이해하기

API(Application Programming Interface)는 프로그램끼리 서로 데이터를 주고받거나 기능을 사용하기 위해 미리 정해놓은 소통 방법이자 약속입니다. 모든 API는 '요청(Request)'과 '응답(Response)'이라는 두 가지 핵심 개념으로 작동합니다.

뤼튼의 예:

  1. 사용자가 뤼튼 앱에 "AI에 대해 설명하는 시를 써줘" 라고 입력합니다.
  2. 뤼튼 프로그램은 이 내용을 OpenAI가 정해놓은 API 규칙에 맞춰 OpenAI 서버로 ‘요청’합니다.
  3. OpenAI의 AI 모델이 시를 생성한 뒤, 그 결과물을 다시 뤼튼 프로그램으로 ‘응답’해 줍니다.
  4. 뤼튼은 전달받은 시를 앱 화면에 표시합니다.

뤼튼의 핵심 경쟁력은 빌려온 AI 엔진을 한국 시장에 맞게 튜닝하는 기술에 있습니다.

  • 파인튜닝 (Fine-tuning): OpenAI가 만든 '만물박사 AI'에게, 뤼튼이 엄선한 양질의 한국어 데이터를 추가로 집중 학습시켜 '한국어 전문가'로 만듭니다.
  • 검색 증강 생성 (RAG): 사용자의 질문에 답변하기 전 먼저 최신 웹 정보를 실시간으로 검색하고, 이 검색 결과를 '참고 자료'로 AI에게 함께 전달하여 더 정확하고 시의성 있는 답변을 생성하게 만듭니다.
  • 전략적 제휴와 모델 최적화: OpenAI의 GPT뿐만 아니라, 네이버의 '하이퍼클로바X' 와 같이 한국어에 특화된 국내 모델 API도 함께 사용하여 질문 종류에 따라 가장 적합한 AI 모델을 선택합니다.
