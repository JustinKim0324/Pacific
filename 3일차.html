GEMS (Gemini) 만들기

반복 작업을 위한 '나만의 특화 AI' 기능입니다. 짧은 질문으로 일관성 있는 결과물을 얻으세요.

1

사이드바 'GEMS' → 'GEMS 탐색하기' 클릭

2

`+새 Gem` 클릭 후 이름, 요청사항 입력

3

'지식' 란에 파일 업로드로 전문성 부여

4

미리보기로 테스트 후 '저장'하면 완성!

공유 정책

현재는 Google이 제공하는 GEMS만 사용 가능하며, 유저간 공유는 지원되지 않습니다.

GPTS (ChatGPT) 만들기

대화형으로 나만의 맞춤형 GPT를 만들고, 다른 사람들과 공유할 수 있는 강력한 기능입니다.

1

사이드바 'GPT' → `+만들기` 클릭

2

채팅 대화로 이름, 지침, 설정 등 입력

3

필요 시 '첨부파일'로 기초 지식 학습

4

오른쪽 화면에서 테스트 후 '만들기' 클릭

공유 정책

내가 만든 GPTS를 다른 사용자들에게 공유하고, 다른 사람이 만든 GPTS도 사용 가능합니다.

AI 코딩: 아이디어를 현실로

홈페이지 만들어보기

HTML(뼈대), CSS(디자인), JS(동작)를 몰라도 AI에게 설명하면 웹페이지 코드를 완성해줍니다.

1. 프롬프트 입력: '나태주 시인 홈페이지 만들어줘'

2. 코드 생성: AI가 전체 코드를 작성

3. 파일 저장: 메모장에 붙여넣고 `.html`로 저장

4. 웹 호스팅: 서비스를 이용해 전 세계에 공개!

인포그래픽 만들어보기

복잡한 정보(Information)와 데이터(Data)를 시각적인 그래픽(Graphic)으로 표현하는 콘텐츠입니다.

1. 데이터 준비: 시각화할 내용을 준비

2. AI 요청: '만들기' 버튼 등으로 인포그래픽 생성

3. 코드 활용: 생성된 코드를 `.html` 파일로 저장

4. 결과 확인: 브라우저에서 시각화된 결과 확인

AI 서비스의 비밀: API와 뤼튼(Wrtn) 사례

API란? "프로그램 사이의 소통 규칙"

AI 모델을 직접 만들지 않아도, API를 통해 AI 기능을 '빌려와' 서비스에 포함시킬 수 있습니다.

요청(Request)

"AI에 대해 설명하는 시를 써줘"

응답(Response)

"차가운 논리의 숲에서, 따스한 빛을 찾는 여정..." (생성된 시)

뤼튼(Wrtn)이 한국어를 잘하는 비결

단순히 API를 쓰는 것을 넘어, 빌려온 AI 엔진을 한국 시장에 맞게 '튜닝'하는 기술이 핵심입니다.

① 파인튜닝 (Fine-tuning)

엄선된 한국어 데이터(신조어, 뉴스 등)를 추가 학습시켜 '한국어 전문가'로 만듭니다.

② 검색 증강 생성 (RAG)

'오픈북 시험'처럼, 답변 전 최신 웹 정보를 실시간으로 검색하여 참고합니다.

③ 전략적 제휴와 모델 최적화

질문에 가장 적합한 AI 모델(GPT, 하이퍼클로바X 등)을 순간적으로 선택하여 사용합니다.
